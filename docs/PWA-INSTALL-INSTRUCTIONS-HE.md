# הוספה למסך הבית — Quant AI (PWA)

המערכת פועלת כ־PWA (Progressive Web App) וניתן להתקין אותה על המכשיר כמו אפליקציה, בלי פס הכתובת והכפתורים של הדפדפן.

---

## אייפון (iPhone) — Safari

1. **פתח את האתר ב־Safari**  
   יש להיכנס לכתובת האתר רק בדפדפן Safari (לא Chrome או אחר).

2. **לחץ על כפתור "שתף"**  
   בתחתית המסך (או בראש המסך) — אייקון הריבוע עם החץ המכוון למעלה.

3. **בחר "הוסף למסך הבית"**  
   גלול ברשימה ובחר באפשרות **"הוסף למסך הבית"** (Add to Home Screen).

4. **אשר את השם ולחץ "הוסף"**  
   ניתן לשנות את שם האפליקציה (למשל "Quant AI") ואז ללחוץ **"הוסף"** בפינה הימנית העליונה.

5. **האפליקציה תופיע במסך הבית**  
   לחיצה על האייקון תפתח את המערכת במצב מסך מלא, בלי ממשק Safari.

**טיפ:** אם האפשרות "הוסף למסך הבית" לא מופיעה, וודא שאתה ב־Safari ולא בדפדפן אחר.

---

## אנדרואיד (Android) — Chrome

1. **פתח את האתר ב־Chrome**  
   גלוש לאתר באמצעות דפדפן Chrome.

2. **הפעל את תפריט הדפדפן**  
   שלוש נקודות אנכיות (⋮) בפינה הימנית העליונה.

3. **בחר "התקן אפליקציה" או "הוסף למסך הבית"**  
   בתפריט חפש **"התקן אפליקציה"** (Install app) או **"הוסף למסך הבית"** (Add to Home screen) ולחץ עליה.

4. **אשר בהתקנה**  
   יופיע חלון עם שם האפליקציה ואייקון — אשר עם **"התקן"** (Install).

5. **האפליקציה תותקן ותופיע בתפריט האפליקציות ובמסך הבית**  
   פתיחה דרך האייקון תציג את המערכת במסך מלא, כמו אפליקציה native.

**טיפ:** אם "התקן אפליקציה" לא מופיע, ייתכן שהדפדפן דורש ביקור נוסף באתר או חיבור אינטרנט יציב. וודא שהאתר נטען ב־HTTPS.

---

## דרישות כלליות

- **חיבור HTTPS** — האתר חייב להיטען בכתובת מאובטחת (https).
- **אייקונים** — אם הוגדרו אייקונים בתיקיית `public/icons` (icon-192.png, icon-512.png), יוצג האייקון של האפליקציה. אם לא, ייתכן שיוצג אייקון ברירת מחדל.
- **Safari באייפון** — חובה להשתמש ב־Safari כדי שהאפשרות "הוסף למסך הבית" תופיע.

---

*לאחר ההתקנה, האפליקציה תפעל במצב standalone (מסך מלא) עם רקע וצבע שורת הסטטוס בהתאם להגדרות המערכת.*
