# PWA Icons

Place your PWA icons here so "Add to Home Screen" works correctly:

- **icon-192.png** — 192×192 px (required for Android and some launchers)
- **icon-512.png** — 512×512 px (required for splash screens and high-DPI)

Use a square image with your app logo; transparent or solid background is fine.  
If these files are missing, the app may still be installable but will show a default or placeholder icon.
